# Домашнее задание: Линейные модели

Реализация градиентного спуска, генератора батчей и логистической регрессии с Elastic Net регуляризацией с нуля.

## Основные компоненты

- **Градиентный спуск** — аналитическая реализация для визуализации сходимости
- **Генератор батчей** — с перемешиванием и отбрасыванием последнего неполного батча
- **`MyLogisticRegression`** — градиент, обучение SGD, предсказание вероятностей
- **`MyElasticLogisticRegression`** — наследование с добавлением L1/L2 регуляризации

## Технологии

`Python` `NumPy` `scikit-learn` `Matplotlib` `Plotly`

## Запуск

Открыть в Colab или локально:

```bash
jupyter notebook hw_2_linear_models.ipynb
```

*Примечание: файл `train.csv` для MNIST требуется загрузить вручную.*
