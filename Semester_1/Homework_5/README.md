# 🎨 Классификация персонажей «Симпсонов» с помощью CNN и Transfer Learning

Домашняя работа по курсу **«Глубокое обучение»** (Deep Learning School, МФТИ).

В рамках проекта была решена задача многоклассовой классификации изображений персонажей мультсериала «Симпсоны». Для повышения качества использовались как собственная сверточная нейронная сеть, так и подход **Transfer Learning** на основе предобученной модели ResNet18.

## 📌 Цель

Построить модель компьютерного зрения, способную по изображению определить персонажа сериала среди десятков различных классов.

Основные задачи:

* подготовка и обработка изображений;
* работа с `Dataset` и `DataLoader`;
* реализация сверточной нейронной сети на PyTorch;
* обучение и оценка модели;
* применение Transfer Learning;
* формирование предсказаний для тестовой выборки.

---

## 🧹 Основные этапы работы

### 1. Подготовка данных

* Загрузка датасета изображений персонажей.
* Кодирование классов через `LabelEncoder`.
* Разделение данных на обучающую и валидационную выборки со стратификацией.
* Создание собственного класса `SimpsonsDataset`.
* Формирование `DataLoader` для эффективной загрузки данных.

### 2. Предобработка изображений

Использованы стандартные инструменты `torchvision.transforms`:

* изменение размера изображений;
* преобразование в тензоры;
* нормализация каналов RGB;
* подготовка батчей для обучения на GPU.

### 3. Базовая сверточная нейронная сеть

Реализована CNN для классификации персонажей:

* сверточные слои (`Conv2D`);
* функции активации ReLU;
* операции подвыборки (`MaxPooling`);
* полносвязные слои классификатора;
* оптимизация через Adam.

Для оценки качества использовались:

* Accuracy;
* F1-score;
* анализ динамики обучения на валидационной выборке.

### 4. Transfer Learning с ResNet18

Для улучшения качества была использована предобученная модель **ResNet18**:

* загрузка весов, обученных на ImageNet;
* заморозка части слоев;
* замена классификационной головы под количество классов датасета;
* дообучение на изображениях персонажей.

Такой подход позволил существенно повысить качество классификации по сравнению с базовой CNN.

### 5. Формирование Kaggle Submission

После обучения лучшей модели были получены предсказания на тестовом наборе данных и сформирован файл:

`transfer_learning_resnet.csv`

готовый для загрузки на платформу Kaggle.
