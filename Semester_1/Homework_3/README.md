# 🗡️ Прогноз оттока клиентов (Churn Prediction)

Домашняя работа по курсу **«Классическое машинное обучение и компьютерное зрение»** (Deep Learning School, Stepik).  
Это решение задачи бинарной классификации с платформы Kaggle. Цель — предсказать, прекратит ли клиент пользоваться услугами компании (Churn).

## 📌 Цель
Построить и обучить модель машинного обучения, которая на основе демографических данных, информации об услугах и платежном поведении клиента определяет вероятность его оттока.

## 🧹 Основные этапы работы
- **EDA и очистка данных** — Обработка пропусков (замена на `NaN` с последующим заполнением медианой/категорией `'Unknown'`), анализ распределений и классовый баланс.
- **Предобработка и пайплайн** — Создание `ColumnTransformer` для стандартизации числовых признаков и One-Hot Encoding категориальных признаков.
- **Моделирование** — Обучение `LogisticRegression` с подбором гиперпараметра `C` через `GridSearchCV` (кросс-валидация).
- **Улучшение решения** — Обучение `CatBoostClassifier` с подбором гиперпараметров (`iterations`, `learning_rate`, `depth`) и функцией ранней остановки.

## 🚀 Результат
Лучшая модель (`CatBoostClassifier`) предсказывает отток клиента с метрикой **ROC-AUC ≈ 0.857** на валидационной выборке.
Финальный файл `my_submission.csv` с вероятностями классов на тестовой выборке готов к загрузке.

## 🛠 Стек
`pandas`, `matplotlib`, `seaborn`, `scikit-learn` (Pipeline, ColumnTransformer, StandardScaler, OneHotEncoder, LogisticRegression, GridSearchCV), `catboost`.

## 📂 Содержимое
- `3_hw_kaggle.ipynb` — Jupyter Notebook с полным пайплайном: от загрузки данных до создания сабмишна.
- `my_submission.csv` — Файл предсказаний для тестовой выборки.

## 💡 Вывод
Градиентный бустинг (CatBoost) на табличных данных с категориальными признаками показывает результат, превосходящий линейные модели, даже без глубокой ручной предобработки признаков.
