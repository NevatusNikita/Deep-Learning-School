# 🧠 Семантическая сегментация медицинских изображений

Задача — построить модели для сегментации поражений кожи (меланома и невусы) на дерматоскопических снимках.

---

## 📌 Цель

Разработать нейронную сеть, которая по фотографии кожного образования точно выделяет область поражения (pixel-wise segmentation).  
Метрика качества — **Dice coefficient** (основная) и **IoU**.

## 🧹 Основные этапы работы

- **Подготовка данных** — загрузка и разархивация PH2 Dataset, приведение изображений к размеру 256×256
- **Бейзлайн** — реализация SegNet + BCE Loss, обучение и валидация
- **Эксперименты с лоссами** — Dice Loss, Focal Loss, комбинированные функции потерь
- **Улучшенная архитектура** — реализация **UNet**
- **Аугментации и тюнинг** — улучшение качества модели
- **BONUS** — эксперименты с современными loss-функциями из статей (опционально)

## 🚀 Результаты

| Модель       | Loss              | Dice (val) | IoU (val) |
|--------------|-------------------|------------|-----------|
| SegNet       | BCE               | 0.872      | 0.781     |
| SegNet       | Dice + BCE        | 0.891      | 0.812     |
| **UNet**     | **Dice + Focal**  | **0.923**  | **0.862** |

**Лучший результат:** Dice = **0.923** на валидационной выборке.

## 💡 Выводы

- UNet значительно превосходит SegNet на данной задаче.
- Комбинация **Dice + Focal Loss** даёт лучшее качество по сравнению с чистым BCE.
- Качественные аугментации (особенно geometric + color jitter) критически важны для медицинской сегментации.
- Даже на небольшом датасете (200 изображений) с правильным подходом можно добиться высокого качества.
