# ML Homework: Deep Learning School (Stepik)

Репозиторий с домашними заданиями по курсу **«Первая часть: классическое машинное обучение и компьютерное зрение»** от [Deep Learning School](https://stepik.org/course/272073/syllabus?search=9401731520) на Stepik.

## 📚 Содержание курса

В рамках курса рассматриваются:

- **Классическое ML**: линейная и логистическая регрессия, деревья решений, ансамбли (Random Forest, Gradient Boosting)
- **Компьютерное зрение**: основы работы с изображениями, фильтры, детекторы, сегментация.
