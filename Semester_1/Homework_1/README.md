# 🗡️ Прогноз выживания персонажей «Игры Престолов»

Домашняя работа по курсу **«Классическое машинное обучение и компьютерное зрение»** (Deep Learning School, Stepik).  
Задача — предсказать, выживет ли персонаж в мире «Песни Льда и Огня», используя реальные данные из вики-проекта.

## 📌 Цель
Построить модель классификации, которая на основе социального статуса, родственных связей, популярности на вики и других признаков определяет исход персонажа.

## 🧹 Основные этапы работы
- **EDA и очистка данных** — обработка пропусков, создание признаков `boolDeadRelations`, `age_value`/`age_no_data`.
- **Feature engineering** — группировка культур (`Northmen`, «Essos Nations», etc.), унификация титулов и домов, One-Hot Encoding.
- **Моделирование** — обучение `LogisticRegression` с подбором гиперпараметров (GridSearchCV).
- **Оценка** — метрика `accuracy`; итоговый результат на валидации ≈ **0.8013**.

## 🚀 Результат
Модель предсказывает выживание персонажа с точностью **~80%**.  
Финальный файл `new_submission.csv` с предсказаниями на тестовой выборке готов к загрузке.

## 🛠 Стек
`pandas`, `matplotlib`, `seaborn`, `scikit-learn` (OneHotEncoder, StandardScaler, LogisticRegression, GridSearchCV).

## 📂 Содержимое
- `1_hw_game_of_thrones.ipynb` — полный пайплайн: от загрузки данных до создания сабмишна.
- Обработка категориальных признаков с редкими значениями и добавлением категории `no_data`.

## 💡 Вывод
Feature engineering и правильная обработка пропусков критически важны для качества модели в задачах с реальными (иногда сильно разреженными) данными.
