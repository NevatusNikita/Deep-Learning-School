# 🧠 Полносвязные и свёрточные нейронные сети на PyTorch

Домашняя работа по курсу **«Глубокое обучение»** (Deep Learning School, МФТИ).

В рамках проекта реализованы базовые компоненты обучения нейронных сетей на **PyTorch**: от логистической регрессии и полносвязных архитектур до классической сверточной сети **LeNet** для распознавания изображений.

## 📌 Цель

Изучить полный цикл разработки нейронных сетей:

* подготовка данных и создание `Dataset` / `DataLoader`;
* реализация моделей на PyTorch;
* написание собственных циклов обучения и валидации;
* сравнение различных функций активации;
* применение сверточных нейронных сетей для задач компьютерного зрения.

---

## 🧹 Основные этапы работы

### 1. Классификация датасета Moons

* Генерация синтетического датасета `make_moons`;
* Подготовка данных и создание загрузчиков;
* Реализация логистической регрессии через `nn.Module`;
* Написание универсальной функции обучения;
* Визуализация разделяющей поверхности и анализ качества модели.

**Результат:** accuracy ≈ **0.89** на валидационной выборке.

### 2. Полносвязная нейронная сеть для MNIST

* Загрузка и подготовка датасета рукописных цифр;
* Построение MLP на основе `nn.Sequential`;
* Обучение и валидация модели;
* Исследование влияния различных функций активации:

  * ReLU
  * LeakyReLU
  * ELU
  * GELU
  * Tanh

### 3. Сверточная нейронная сеть (CNN)

* Изучение принципов работы свёрток и ядер фильтрации;
* Реализация архитектуры **LeNet**;
* Обучение CNN на датасете MNIST;
* Сравнение качества с полносвязной архитектурой.

**Результат:** accuracy ≈ **0.97** на валидационной выборке.

---

## 🚀 Полученные результаты

| Модель                  | Датасет | Accuracy                       |
| ----------------------- | ------- | ------------------------------ |
| Логистическая регрессия | Moons   | ~0.89                          |
| Полносвязная сеть (MLP) | MNIST   | Высокое качество классификации |
| LeNet (CNN)             | MNIST   | ~0.97                          |

Главный вывод — сверточные нейронные сети значительно эффективнее используют пространственную структуру изображений и обеспечивают более высокое качество по сравнению с полносвязными архитектурами.
